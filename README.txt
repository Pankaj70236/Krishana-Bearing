# Krishna Bearing Agency Website (Static)

## Run locally
- Just open `index.html` in Chrome.
- Or use VS Code -> Live Server.

## Files
- `index.html` (Home) uses `assets/bg-shoe.png` background and blur circle effect.
- `products.html` uses `assets/bg-parts.png` background.
- `pages/*.html` category pages (brands + WhatsApp order).
- `style.css` + `script.js`

## WhatsApp / Call
Numbers used:
- 9928857705
- 7023622705

Change them in `script.js` or pages if needed.

## Replace brand logos
Currently logos are self-made SVG placeholders in `assets/brand-*.svg`.
When you give real brand logos, replace those files (keep same filename) and refresh.

## Item Images
- Put your images in `assets/items/`
- Update `assets/items.json` (list filenames per category)
- Best open with Live Server so `fetch()` works.
